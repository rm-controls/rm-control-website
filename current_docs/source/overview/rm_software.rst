RM-Software
=================
RM-Software 是一套适用于RoboMaster机器人的控制解决方案及软件栈， 其最大特点是无单片机电控，有着以下优势：

1. 极其简单的系统结构

   * 不需要编写单片机程序
   * 主控与视觉识别的计算设备相同
2. 极高开发效率

   * 所有电机, 传感器数据均可记录和可视化
   * 所有控制参数均可动态和远程调试
   * 丰富的成熟软件包
3. 兼容性可复用性强

   * 提供三套硬件解决方案
   * 程序模块化, 并通过ROS plugins进行动态加载, 所有机器人共用一套代码
   * 支持 CAN 总线, DBus, UART, GPIO, PWM 等 RM 常用外设
4. 相比单片机而言, 极高的算力

Note

 无单片机电控是指使用机载PC代替单片机作为主控制器; 电机驱动等设备还是使用了单片机.
